<h1>ListItem</h1>

## Description

A ListItem contains the content used in an [OrderedList](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-lists-orderedlist--docs) or [UnorderedList](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-lists-unorderedlist--docs).

The text color can be customized with the `swatch` prop and a [swatch color](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors).

## Usage

```jsx
import { ListItem } from 'react-pdf-builder';
```

```jsx
<ListItem>List item</ListItem>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop   | Type          | Description                                                                                                                                                                  |
| ------ | ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| swatch | `SwatchColor` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |

```jsx
import * as React from 'react';
import { UnorderedList, OrderedList, ListItem, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <UnorderedList>
            <ListItem>First list item</ListItem>
            <ListItem>Second list item</ListItem>
            <ListItem>Third list item</ListItem>
          </UnorderedList>
          <OrderedList>
            <ListItem>First list item</ListItem>
            <ListItem>Second list item</ListItem>
            <ListItem>Third list item</ListItem>
          </OrderedList>
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
