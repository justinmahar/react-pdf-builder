<h1>Button</h1>

## Description

The Button component is an interactive element that users can click on. 

By specifying the `href` prop, the button can link to any URL of your choice. 

You can customize the look with [swatch colors](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors), styles, and [class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names). In addition to the theme's [swatch colors](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors), a special swatch is available called `'link'` that will make the button appear as a link.

## Usage

```jsx
import { Button } from 'react-pdf-builder';
```

```jsx
<Button href="https://justinmahar.github.io/react-pdf-builder/">Click Me</Button>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop      | Type                          | Description                                                                                                                                                                                                  |
| --------- | ----------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| href      | `string`                      | Optional. When provided, Button becomes a clickable link.                                                                                                                                                    |
| pill      | `boolean`                     | Optional. When `true`, corners will be round.                                                                                                                                                                |
| linkColor | `string`                      | Optional. Color for the button text when the `link` swatch is used.                                                                                                                                          |
| swatch    | `SwatchColor` &#124; `'link'` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string, or the special `'link'` swatch. |
