<h1>Box</h1>

## Description

Description of the Box component.

## Usage

```jsx
import { Box } from 'react-pdf-builder';
```

Describe how to use the component here.

```jsx
<Box dir="y" className="gap-2">
  <Box>Item 1</Box>
  <Box>Item 2</Box>
</Box>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop    | Type                                                | Description |
| ------- | --------------------------------------------------- | ----------- |
| dir     | `'row'` &#124; `'x'` &#124; `'column'` &#124; `'y'` | Optional.   |
| grow    | `boolean`                                           | Optional.   |
| shrink  | `boolean`                                           | Optional.   |
| gap     | `number`                                            | Optional.   |
| padding | `number`                                            | Optional.   |
