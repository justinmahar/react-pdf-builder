<h1>Box</h1>

## Description

The Box component helps you lay out and organize child components effortlessly. 

It allows you to arrange items in familiar Flexbox rows or columns, and provides options for growth, shrinking, and padding, and gap.

The default direction is `row` (or `x`).

## Usage

```jsx
import { Box } from 'react-pdf-builder';
```

```jsx
<Box dir="y" className="gap-2">
  <Box>Item 1</Box>
  <Box>Item 2</Box>
</Box>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop    | Type                                                | Description                                                                                                                                   |
| ------- | --------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| dir     | `'row'` &#124; `'x'` &#124; `'column'` &#124; `'y'` | Optional. The flex direction. Use `row` or `x` for horizontal row layout. Use `column` or `y` for vertical column layout. Default is `'row'`. |
| grow    | `boolean`                                           | Optional. When `true`, the Box will expand within its parent Box.                                                                             |
| shrink  | `boolean`                                           | Optional. When `true`, the Box will shrink within its parent Box.                                                                             |
| gap     | `number`                                            | Optional. Convenient way to quickly set the `gap` style.                                                                                      |
| padding | `number`                                            | Optional. Convenient way to quickly set the `padding` style.                                                                                  |
