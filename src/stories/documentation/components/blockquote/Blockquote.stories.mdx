<h1>Blockquote</h1>

## Description

The Blockquote component is used to display a section of quoted text, styled to stand out from the surrounding content.

By using the `swatch` prop with a [swatch color](http://localhost:6006/?path=/docs/documentation-themes--docs#swatch-colors), you can easily customize the background and border colors.

## Usage

```jsx
import { Blockquote } from 'react-pdf-builder';
```

```jsx
<Blockquote className="lh-base">
  “Contents of blockquote go here.”
</Blockquote>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop          | Type          | Description                                                                                                                                                                  |
| ------------- | ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| swatch        | `SwatchColor` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |
| swatchOpacity | `number`      | Optional.                                                                                                                                                                    |

import * as React from 'react';

      <Document>

          <Blockquote className="lh-base">

          </Blockquote>

            Blockquotes can be swatched easily by passing the swatch prop. This blockquote is swatched "blue".
          </Blockquote>
          <Blockquote className="lh-base" swatch="yellow">
            Blockquotes can be swatched easily by passing the swatch prop. This blockquote is swatched "yellow".
          </Blockquote>
          <Blockquote className="lh-base" swatch="green">

          </Blockquote>
        </ThemedPage>
      </Document>
