<h1>CircleShape</h1>

## Description

The CircleShape component lets you easily create and customize SVG circles for your PDF. It's an [EllipseShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-ellipseshape--docs) with the same width and height.

Just provide a `size` and then configure your circle via the props for [EllipseShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-ellipseshape--docs).

## Usage

```jsx
import { CircleShape } from 'react-pdf-builder';
```

```jsx
<CircleShape size={300} />
```

### Props

| Inherits all props from                                                                                                        |
| ------------------------------------------------------------------------------------------------------------------------------ |
| [EllipseShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-ellipseshape--docs) |
| [Svg](https://react-pdf.org/svg#svg)                                                                                           |

And supports the following props:

| Prop | Type     | Description |
| ---- | -------- | ----------- |
| size | `number` | Optional.   |

## CircleShape Example

<DocsPDF>
  <CircleShape 
    size={300}
    gradient={['red', 'blue']}
    gradientType={GradientType.leftToRight}
    ellipseProps={{ stroke: 'darkblue', strokeWidth: 5 }}
   />

```jsx
import * as React from 'react';
import { CircleShape, GradientType, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <CircleShape 
            size={300}
            gradient={['red', 'blue']}
            gradientType={GradientType.leftToRight}
            ellipseProps={{ stroke: 'darkblue', strokeWidth: 5 }}
          />
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
