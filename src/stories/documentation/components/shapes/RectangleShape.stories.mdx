<h1>RectangleShape</h1>

## Description

The RectangleShape component lets you easily create and customize SVG rectangles for your PDF.

Just provide a `width` and `height` then configure your rectangle via the props below.

## Usage

```jsx
import { RectangleShape } from 'react-pdf-builder';
```

```jsx
<RectangleShape width={300} height={200} />
```

### Props

| Inherits all props from              |
| ------------------------------------ |
| [Svg](https://react-pdf.org/svg#svg) |

And supports the following props:

| Prop                 | Type                                                 | Description                                                                                                                                                                  |
| -------------------- | ---------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| rectProps            | `Partial<RectProps>`                                 | Optional.                                                                                                                                                                    |
| linearGradientProps  | `Partial<LinearGradientProps>`                       | Optional.                                                                                                                                                                    |
| radialGradientProps  | `Partial<RadialGradientProps>`                       | Optional.                                                                                                                                                                    |
| gradient             | `GradientStop[]` &#124; `string[]`                   | Optional.                                                                                                                                                                    |
| gradientType         | `GradientType`                                       | Optional.                                                                                                                                                                    |
| linearGradientCoords | `{ x1: number; y1: number; x2: number; y2: number }` | Optional.                                                                                                                                                                    |
| radialGradientCoords | `{ x: number; y: number }`                           | Optional.                                                                                                                                                                    |
| className            | `string`                                             | Optional.                                                                                                                                                                    |

### Gradient Types 

The following gradient types are available:

- `GradientType.leftToRight`
- `GradientType.rightToLeft`
- `GradientType.topToBottom`
- `GradientType.bottomToTop`
- `GradientType.topLeftToBottomRight`
- `GradientType.bottomLeftToTopRight`
- `GradientType.bottomRightToTopLeft`
- `GradientType.topRightToBottomLeft`
- `GradientType.radial`

### Gradient Stops

You can use gradient stops to create custom gradients.

A gradient stop is an object with the following properties:

| Property    | Type                     | Description                                                 |
| ----------- | ------------------------ | ----------------------------------------------------------- |
| offset      | `string` &#124; `number` | Offset as a percentage, such as `0%`, `50%`, `80%`, `100%`. |
| stopColor   | `string`                 | The color for the stop.                                     |
| stopOpacity | `string` &#124; `number` | Opacity for the stop from `0` to `1`.                       |

For example, here is a gradient stop with an offset of 20% with a gray color and an opacity of 4%:

```ts
{ offset: '20%', stopColor: '#5f5f5f', stopOpacity: 0.04 }
```

## RectangleShape Example

<DocsPDF>
  <RectangleShape 
    width={300}
    height={200}
    gradient={['red', 'blue']}
    gradientType={GradientType.leftToRight}
    rectProps={{ stroke: 'darkblue', strokeWidth: 10 }}
   />
</DocsPDF>

```jsx
import * as React from 'react';
import { RectangleShape, GradientType, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <RectangleShape 
            width={300}
            height={200}
            gradient={['red', 'blue']}
            gradientType={GradientType.leftToRight}
            rectProps={{ stroke: 'darkblue', strokeWidth: 10 }}
          />
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
