<h1>EllipseShape</h1>

## Description

The EllipseShape component lets you easily create and customize SVG ellipses for your PDF.

Just provide a `width` and `height` then configure your ellipse via the props below.

## Usage

```jsx
import { EllipseShape } from 'react-pdf-builder';
```

```jsx
<EllipseShape width={300} height={200} />
```

### Props

| Inherits all props from              |
| ------------------------------------ |
| [Svg](https://react-pdf.org/svg#svg) |

And supports the following props:

| Prop                 | Type                                                                       | Description                                                                                                                                                                                                              |
| -------------------- | -------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| ellipseProps         | [`Partial<EllipseProps>`](https://react-pdf.org/svg#ellipse)               | Optional. Props for the underlying SVG [Ellipse](https://react-pdf.org/svg#ellipse). Specify a stroke here, such as `{ stroke: 'darkblue', strokeWidth: 5 }`.                                                            |
| linearGradientProps  | [`Partial<LinearGradientProps>`](https://react-pdf.org/svg#lineargradient) | Optional. Props for the underlying SVG [LinearGradient](https://react-pdf.org/svg#lineargradient). Use for advanced gradient customization.                                                                              |
| radialGradientProps  | [`Partial<RadialGradientProps>`](https://react-pdf.org/svg#radialgradient) | Optional. Props for the underlying SVG [RadialGradient](https://react-pdf.org/svg#radialgradient). Use for advanced gradient customization.                                                                              |
| gradient             | `string[]` &#124; `GradientStop[]`                                         | Optional. An array of custom gradient colors, or [gradient stops](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-gradientbackdrop--docs#gradient-stops).                           |
| gradientType         | `GradientType`                                                             | Optional. The [type of gradient](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-gradientbackdrop--docs#gradient-types). Defaults to a linear gradient, `GradientType.topToBottom`. |
| linearGradientCoords | `{ x1: number; y1: number; x2: number; y2: number }`                       | Optional. Specify two points defining the direction of the gradient. Values are normalized from `0` to `1`, `0` being left/top, `1` being right/bottom. Overrides the direction specified in the `gradientType` prop.    |
| radialGradientCoords | `{ x: number; y: number }`                                                 | Optional. Specify the location of the radial gradient. Values are normalized from `0` to `1`, `0` being left/top, `1` being right/bottom. Use in conjunction with a `gradientType` of `GradientType.radial`.             |
| className            | `string` &#124; `undefined`                                                | Optional. Class name string used to style the component. [Class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names) are defined in themes.                        |

### Gradient Types 

The following gradient types are available:

- `GradientType.leftToRight`
- `GradientType.rightToLeft`
- `GradientType.topToBottom`
- `GradientType.bottomToTop`
- `GradientType.topLeftToBottomRight`
- `GradientType.bottomLeftToTopRight`
- `GradientType.bottomRightToTopLeft`
- `GradientType.topRightToBottomLeft`
- `GradientType.radial`

### Gradient Stops

You can use gradient stops to create custom gradients.

A gradient stop is an object with the following properties:

| Property    | Type                     | Description                                                 |
| ----------- | ------------------------ | ----------------------------------------------------------- |
| offset      | `string` &#124; `number` | Offset as a percentage, such as `0%`, `50%`, `80%`, `100%`. |
| stopColor   | `string`                 | The color for the stop.                                     |
| stopOpacity | `string` &#124; `number` | Opacity for the stop from `0` to `1`.                       |

For example, here is a gradient stop with an offset of 20% with a gray color and an opacity of 4%:

```ts
{ offset: '20%', stopColor: '#5f5f5f', stopOpacity: 0.04 }
```

<DocsPDF>
  <EllipseShape

```jsx
import * as React from 'react';
import { EllipseShape, GradientType, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <EllipseShape 
            width={300}
            height={200}
            gradient={['red', 'blue']}
            gradientType={GradientType.leftToRight}
            ellipseProps={{ stroke: 'darkblue', strokeWidth: 5 }}
          />
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
