<h1>SquareShape</h1>

## Description

The SquareShape component lets you easily create and customize SVG squares for your PDF. It's a [RectangleShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-rectangleshape--docs) with the same width and height.

Just provide a `size` and then configure your square via the props for [RectangleShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-rectangleshape--docs).

## Usage

```jsx
import { SquareShape } from 'react-pdf-builder';
```

```jsx
<SquareShape size={300} />
```

### Props

| Inherits all props from                                                                                                            |
| ---------------------------------------------------------------------------------------------------------------------------------- |
| [RectangleShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-rectangleshape--docs) |
| [Svg](https://react-pdf.org/svg#svg)                                                                                               |

And supports the following props:

| Prop | Type     | Description                                     |
| ---- | -------- | ----------------------------------------------- |
| size | `number` | Optional. Width and height of square in points. |
  
## SquareShape Example

<DocsPDF>
  <SquareShape 
    size={300}

    rectProps={{ stroke: 'darkblue', strokeWidth: 10 }}

```jsx
import * as React from 'react';
import { SquareShape, GradientType, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <SquareShape 
            size={300}
            gradient={['red', 'blue']}
            gradientType={GradientType.leftToRight}
            rectProps={{ stroke: 'darkblue', strokeWidth: 10 }}
          />
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
