<h1>SquareShape</h1>

## Description

Description of the SquareShape component.

## Usage

```jsx
import { SquareShape } from 'react-pdf-builder';
```

```jsx
<SquareShape size={300} />
```

### Props

| Inherits all props from                                                                                                            |
| ---------------------------------------------------------------------------------------------------------------------------------- |
| [RectangleShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-rectangleshape--docs) |
| [Svg](https://react-pdf.org/svg#svg)                                                                                               |

And supports the following props:

| Prop   | Type                             | Description                                                                                                                                                                  |
| ------ | -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| size | `number`                           | Optional.                                                                                                                                                                    |
| swatch | `SwatchColor` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |
