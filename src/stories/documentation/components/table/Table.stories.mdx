<h1>Table</h1>

## Description

The Table component is designed for displaying structured data in a clean and organized format. 

It includes options for borders, striping, and custom column widths.

You can apply [swatch colors](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) to a table via the `swatch` prop. Applying a swatch to a table will affect the striping and border colors.

## Usage

```jsx
import { Table, TableRow, TableCell } from 'react-pdf-builder';
```

```jsx
<Table bordered striped>
  <TableRow>
    <TableCell>Lorem ipsum dolor</TableCell>
    <TableCell>Sit amet</TableCell>
    <TableCell>Sed do eiusmod</TableCell>
  </TableRow>
  <TableRow>
    <TableCell>Lorem ipsum dolor</TableCell>
    <TableCell>Sit amet</TableCell>
    <TableCell>Sed do eiusmod</TableCell>
  </TableRow>
</Table>
```

### Custom columns

You can use the `colWidths` prop to specify custom widths for each column. By default, cell widths are divided evenly.

[TableRow](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-tables-tablerow--docs) can also be given the `colWidths` prop to change the cells widths for just that row. This is particularly useful if you want to use colspans in your table. See the example below.

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Box](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-box--docs)        |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop               | Type                                          | Description                                                                                                                                                                  |
| ------------------ | --------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| striped            | `boolean`                                     | Optional.                                                                                                                                                                    |
| stripeStyle        | `Style`                                       | Optional.                                                                                                                                                                    |
| inverseStriped     | `boolean`                                     | Optional.                                                                                                                                                                    |
| stripeOpacity      | `number`                                      | Optional.                                                                                                                                                                    |
| rowStyle           | `Style`                                       | Optional.                                                                                                                                                                    |
| cellStyle          | `Style`                                       | Optional.                                                                                                                                                                    |
| bordered           | `boolean`                                     | Optional.                                                                                                                                                                    |
| borderedOutside    | `boolean`                                     | Optional.                                                                                                                                                                    |
| borderedInside     | `boolean`                                     | Optional.                                                                                                                                                                    |
| borderedVertical   | `boolean`                                     | Optional.                                                                                                                                                                    |
| borderedHorizontal | `boolean`                                     | Optional.                                                                                                                                                                    |
| borderColor        | `string`                                      | Optional.                                                                                                                                                                    |
| borderStyle        | `'dashed'` &#124; `'dotted'` &#124; `'solid'` | Optional.                                                                                                                                                                    |
| borderWidth        | `string` &#124; `number`                      | Optional.                                                                                                                                                                    |
| colWidths          | `(string OR number)[]`                        | Optional.                                                                                                                                                                    |
| swatch             | `SwatchColor`                                 | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |
