<h1>TableCell</h1>

## Description

The TableCell component is used to define individual cells within a [Table](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-tables-table--docs). 

You can apply [swatch colors](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) to individual cells via the `swatch` prop. Applying a swatch will add a partly transparent background color to the cell.

## Usage

```jsx
import { TableCell } from 'react-pdf-builder';
```

```jsx
<TableCell>Lorem ipsum dolor</TableCell>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Box](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-box--docs)        |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop          | Type          | Description                                                                                                                                                                  |
| ------------- | ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| swatch        | `SwatchColor` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |
| swatchOpacity | `number`      | Optional.                                                                                                                                                                    |
