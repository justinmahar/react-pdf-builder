<h1>Heading6</h1>

## Description

Description of the Heading6 component.

## Usage

```jsx
import { Heading6 } from 'react-pdf-builder';
```

```jsx
<Heading6>Heading 6</Heading6>
```

### Props

| Inherits all props from                                                                                                    |
| -------------------------------------------------------------------------------------------------------------------------- |
| [ThemedText](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedtext--docs) |
| [Text](https://react-pdf.org/components#text)                                                                              |

And supports the following props:

| Prop | Type      | Description                                                                                           |
| ---- | --------- | ----------------------------------------------------------------------------------------------------- |
| rule | `boolean` | Optional. When `true`, a bottom border will be displayed under the heading text. Default `undefined`. |

## Heading6 Example

<DocsPDF>
  <Heading6 className="mb-3">Heading 6</Heading6>
  <Heading6 rule>Heading 6 with rule</Heading6>
</DocsPDF>

```jsx
import * as React from 'react';
import { Heading6, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <Heading6 className="mb-3">Heading 6</Heading6>
          <Heading6 rule>Heading 6 with rule</Heading6>
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
