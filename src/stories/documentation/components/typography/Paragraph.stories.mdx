<h1>Paragraph</h1>

## Description

The Paragraph component formats text into clean, readable paragraphs, making it easy to present information clearly.

Use this component to create structured text sections in your PDF.

By default, Paragraphs have a bottom margin equaling the base font size, and have a line height of `1.5`. 

If a paragraph is the last element, you can easily remove the bottom margin with the class name `mb-0`.

To easily change the line height, the following class names are available for common line heights:

```
lh-1	{ "lineHeight": 1 }
lh-sm	{ "lineHeight": 1.25 }
lh-base	{ "lineHeight": 1.5 }
lh-lg	{ "lineHeight": 2 }
```

## Usage

```jsx
import { Paragraph } from 'react-pdf-builder';
```

```jsx
<Paragraph>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do 
  eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut 
  enim ad minim veniam, quis nostrud exercitation ullamco laboris 
  nisi ut aliquip ex ea commodo consequat.
</Paragraph>
```
