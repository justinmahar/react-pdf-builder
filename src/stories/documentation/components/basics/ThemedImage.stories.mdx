## Description

Description of the ThemedImage component.

## Usage

```jsx
import { ThemedImage } from 'react-pdf-builder';
```

Describe how to use the component here.

```jsx
<ThemedImage src="https://justinmahar.github.io/react-pdf-builder/images/scootie-460.png"/>
```

### Props

| Inherits all props from                         |
| ----------------------------------------------- |
| [Image](https://react-pdf.org/components#image) |

And supports the following props:

| Prop      | Type                        | Description                                                                                                                                                                                       |
| --------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| className | `string` | Optional. Class name string used to style the component. [Class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names) are defined in themes. |

## ThemedImage Example

<DocsPDF>
  <ThemedImage
    className="border p-2 rounded mb-3"
    src="https://justinmahar.github.io/react-pdf-builder/images/scootie-460.png"
    style={{ width: 230 }}
  />
  <Paragraph>This image uses the "border p-2 rounded mb-3" class names from the theme to achieve a thumbnail style with a bottom margin.</Paragraph>
</DocsPDF>

```jsx
import * as React from 'react';
import { ThemedImage, Paragraph, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <ThemedImage
            className="border p-3 rounded mb-3"
            src="https://justinmahar.github.io/react-pdf-builder/images/scootie-460.png"
            style={{ width: 230 }}
          />
          <Paragraph>
            This image uses the "border p-3 rounded mb-3" class names from the theme to achieve a thumbnail style with a
            bottom margin.
          </Paragraph>
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
