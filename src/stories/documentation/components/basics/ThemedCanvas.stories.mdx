## Description

Description of the ThemedCanvas component.

## Usage

```jsx
import { ThemedCanvas } from 'react-pdf-builder';
```

Describe how to use the component here.

```jsx
<ThemedCanvas
  paint={(painter, w, h) => {
    // Paint your canvas here
    return null;
  }}
  style={{ width: 400, height: 400 }}
/>
```

### Props


| Inherits all props from                           |
| ------------------------------------------------- |
| [Canvas](https://react-pdf.org/components#canvas) |

And supports the following props:

| Prop      | Type                        | Description                                                                                                                                                                                       |
| --------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| className | `string` | Optional. Class name string used to style the component. [Class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names) are defined in themes. |

## ThemedCanvas Example

<DocsPDF>
  <ThemedCanvas
    className="border border-2 mb-3"
    paint={(painter, w, h) => {

      painter.fillOpacity(1);
      painter.fontSize(40);
      painter.text('Canvas text', 92, 125);

```jsx
import * as React from 'react';
import { ThemedCanvas, Paragraph, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedCanvas
          className="border border-2 mb-3"
          paint={(painter, w, h) => {
            painter.fillColor('#0d6efd');
            painter.fillOpacity(1);
            painter.fontSize(40);
            painter.text('Canvas text', 92, 125);
            return null;
          }}
          style={{ width: 400, height: 300 }}
        />
        <Paragraph>
          This 400x300 canvas uses "border border-2 mb-3" class names from the theme for a border of width 2, with a
          bottom margin.
        </Paragraph>
      </Document>
    </PDFViewer>
  );
}
```
