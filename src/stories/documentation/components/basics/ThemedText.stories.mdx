## Description

Description of the ThemedText component.

## Usage

```jsx
import { ThemedText } from 'react-pdf-builder';
```

Describe how to use the component here.

```jsx
<ThemedText>This is themed text that supports class names.</ThemedText>
```

### Props

| Inherits all props from                       |
| --------------------------------------------- |
| [Text](https://react-pdf.org/components#text) |

And supports the following props:

| Prop      | Type                             | Description                                                                                                                                                                                       |
| --------- | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| swatch    | `SwatchColor` &#124; `undefined` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string.                      |
| className | `string` &#124; `undefined`      | Optional. Class name string used to style the component. [Class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names) are defined in themes. |

## ThemedText Example

<DocsPDF>
  <ThemedText className="p-3 border">This is themed text with padding and a border from the theme.</ThemedText>
</DocsPDF>

```jsx
import * as React from 'react';
import { ThemedText, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <ThemedText className="p-3 border">This is themed text with padding and a border from the theme.</ThemedText>
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
