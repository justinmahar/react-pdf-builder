### Props

This component inherits all props from [Text](https://react-pdf.org/components#text), with the following additional props:

| Prop      | Type                             | Description                                                                                                                                                                  |
| --------- | -------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| children  | `any`                            | Optional. Can be strings or React components. Strings will automatically be wrapped in [ThemedText](https://justinmahar.github.io/react-pdf-builder/).                       |
| swatch    | `SwatchColor` &#124; `undefined` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |
| className | `string` &#124; `undefined`      | Optional. Class name string used to style the component. Custom class names can be defined in themes.                                                                        |

## ThemedText Example

<DocsPDF>
  <ThemedText className="p-3 border">This is themed text with padding and a border from the theme.</ThemedText>
</DocsPDF>

```jsx
import * as React from 'react';
import { ThemedText, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <ThemedText className="p-3 border">This is themed text with padding and a border from the theme.</ThemedText>
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
