## Description

Description of the Div component.

## Usage

```jsx
import { Div } from 'react-pdf-builder';
```

Describe how to use the component here.

```jsx
<Div>This is a div</Div>
```

### Props

| Inherits all props from                       |
| --------------------------------------------- |
| [View](https://react-pdf.org/components#view) |

And supports the following props:

| Prop      | Type     | Description                                                                                               |
| --------- | -------- | --------------------------------------------------------------------------------------------------------- |
| className | `string` | Optional. A string containing class names defined in the theme, which can be used to style the component. |

## Div Example

<DocsPDF>

```jsx
import * as React from 'react';
import { Div, Paragraph, ThemedText, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <Div className="text-green border border-green p-3 rounded bg-faded-green fw-bold">
            <Paragraph>
              This is a Div with a variety of class names from the theme to achieve a unique look. It contains 3 Paragraph
              components and a ThemedText containing the class names.
            </Paragraph>
            <Paragraph>
              A Div can be used like an HTML div in this framework. It supports both the className and style props to make
              styling easy.
            </Paragraph>
            <Paragraph className="mb-0">Class names used:</Paragraph>
            <ThemedText>"text-green border border-green p-3 rounded bg-faded-green fw-bold"</ThemedText>
          </Div>
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
