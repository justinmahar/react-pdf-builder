
### Props

This component inherits all props from [View](https://react-pdf.org/components#view), with the following additional props:

| Prop      | Type                        | Description                                                                                                                                            |
| --------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------ |
| children  | `any`                       | Optional. Can be strings or React components. Strings will automatically be wrapped in [ThemedText](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedtext--docs). |
| className | `string` &#124; `undefined` | Optional. A string containing class names defined in the theme, which can be used to style the component.                                              |
