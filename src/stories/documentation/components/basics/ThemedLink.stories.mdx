## Description

Description of the ThemedLink component.

## Usage

```jsx

  React PDF Builder
</ThemedLink>
```

Describe how to use the component here.

```jsx

```

### Props


| Inherits all props from                       |
| --------------------------------------------- |
| [Link](https://react-pdf.org/components#link) |

And supports the following props:

| Prop      | Type                             | Description                                                                                                                                                                                       |
| --------- | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| swatch    | `SwatchColor` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string.                      |
| className | `string` &#124; `undefined`      | Optional. Class name string used to style the component. [Class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names) are defined in themes. |
