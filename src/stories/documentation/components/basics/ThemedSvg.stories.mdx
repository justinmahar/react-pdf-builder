## Description

Description of the ThemedSvg component.

## Usage

```jsx
import { ThemedSvg } from 'react-pdf-builder';
```

Describe how to use the component here.

```jsx
<ThemedSvg style={{ width: 220, height: 130 }}>
  <Ellipse cx={110} cy={65} rx={35} ry={35} fill="#9E0024" />
</ThemedSvg>
```

### Props

| Inherits all props from              |
| ------------------------------------ |
| [Svg](https://react-pdf.org/svg#svg) |

And supports the following props:

| Prop      | Type                        | Description                                                                                               |
| --------- | --------------------------- | --------------------------------------------------------------------------------------------------------- |
| className | `string` &#124; `undefined` | Optional. A string containing class names defined in the theme, which can be used to style the component. |
  
## ThemedSvg Example

```jsx
import * as React from 'react';
import { ThemedSvg, Paragraph, ThemedPage } from 'react-pdf-builder';
import { PDFViewer, Document, Ellipse } from '@react-pdf/renderer';

export function Example() {
  return (
    <PDFViewer style={{ width: 420, height: 600 }}>
      <Document>
        <ThemedPage size="LETTER">
          <ThemedSvg className="border border-2 mb-3 bg-white" style={{ width: 220, height: 130 }}>
            <Ellipse cx={110} cy={65} rx={35} ry={35} fill="#9E0024" />
          </ThemedSvg>
          <Paragraph>
            SVG of the Japanese flag, with class name "border border-2 mb-3 bg-white" from the theme to add a border, white
            background, and bottom margin.
          </Paragraph>
        </ThemedPage>
      </Document>
    </PDFViewer>
  );
}
```
