## Description

Description of the ThemedPage component.

## Usage

```jsx
import { ThemedPage } from 'react-pdf-builder';
```

Describe how to use the component here.

```jsx
<ThemedPage size="LETTER">
  Contents of page go here.
</ThemedPage>
```

### Props

| Inherits all props from                       |
| --------------------------------------------- |
| [Page](https://react-pdf.org/components#page) |

And supports the following props:

| Prop      | Type                             | Description                                                                                                                                                                                       |
| --------- | -------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| swatch    | `SwatchColor` &#124; `undefined` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string.                      |
| className | `string` &#124; `undefined`      | Optional. Class name string used to style the component. [Class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names) are defined in themes. |
