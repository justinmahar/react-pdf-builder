<h1>Card</h1>

## Description

The Card component is a flexible container designed to present content in a visually appealing way.

It consists of a [header](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-cards-cardheader--docs) and [body](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-cards-cardbody--docs), making it ideal for showcasing information, images, or any relevant content in a structured format.

You can easily customize the look of your cards with [swatch colors](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) using the `swatch` prop.

## Usage

```jsx
import { Card, CardBody, CardHeader } from 'react-pdf-builder';
```

```jsx
<Card>
  <CardHeader>Header</CardHeader>
  <CardBody className="lh-base">This is the card body.</CardBody>
</Card>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Box](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-box--docs)        |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop          | Type          | Description                                                                                                                                                                  |
| ------------- | ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| swatch        | `SwatchColor` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |
| swatchOpacity | `number`      | Optional. Opacity value for the swatch color, from `0` to `1`.                                                                                                                                                                |
