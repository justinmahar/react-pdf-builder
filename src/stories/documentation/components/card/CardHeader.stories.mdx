<h1>CardHeader</h1>

## Description

The CardHeader component serves as the title or header area for a [Card](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-cards-card--docs).

It helps you clearly label the content within the card. 

You can customize its appearance using various props, including [swatch colors](http://localhost:6006/?path=/docs/documentation-themes--docs#swatch-colors) with the `swatch` prop.

## Usage

```jsx
import { CardHeader } from 'react-pdf-builder';
```

```jsx
<CardHeader>Header</CardHeader>
```

### Props

| Inherits all props from                                                                                      |
| ------------------------------------------------------------------------------------------------------------ |
| [Box](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-box--docs)        |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs) |
| [View](https://react-pdf.org/components#view)                                                                |

And supports the following props:

| Prop          | Type          | Description                                                                                                                                                                  |
| ------------- | ------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| as            | `any`         | Optional.                                                                                                                                                                    |
| noBody        | `boolean`     | Optional.                                                                                                                                                                    |
| withHeader    | `boolean`     | Optional.                                                                                                                                                                    |
| swatch        | `SwatchColor` | Optional. One of the [swatch color names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme, as a string. |
| swatchOpacity | `number`      | Optional.                                                                                                                                                                    |
