## List of Components

### Basics

These components wrap the basic [components](https://react-pdf.org/components) provided by [react-pdf](https://react-pdf.org/).

They all add support for the `className` prop, allowing you to use the [utility classes](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-utility-classes--docs) and even [custom class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#class-names) from the theme to easily style your components.

| Component                                                                                                                      | Description                                                                                                                            |
| ------------------------------------------------------------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------------------------------------------------- |
| [Div](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-div--docs)                   | A flexible container for organizing and styling content, with class name support. Wraps [View](https://react-pdf.org/components#view). |
| [ThemedCanvas](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedcanvas--docs) | A surface for drawing and rendering, with class name support. Wraps [Canvas](https://react-pdf.org/components#canvas).                 |
| [ThemedImage](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedimage--docs)   | A component for displaying images with class name support. Wraps [Image](https://react-pdf.org/components#image).                      |
| [ThemedLink](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedlink--docs)     | A styled hyperlink component for navigation with swatch and class name support. Wraps [Link](https://react-pdf.org/components#link).   |
| [ThemedPage](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedpage--docs)     | A page container with class name support. Wraps [Page](https://react-pdf.org/components#page).                                         |
| [ThemedSvg](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedsvg--docs)       | A container for SVG graphics with class name support. Wraps [Svg](https://react-pdf.org/svg).                                          |
| [ThemedText](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-basics-themedtext--docs)     | A styled text component with class name support. Wraps [Text](https://react-pdf.org/components#text).                                  |

### Typography & Text

Headings, paragraphs, blockquotes, and horizontal rules are available for any text you want to display in your PDF.

| Component                                                                                                                    | Description                                    |
| ---------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------- |
| [Heading1](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-typography-heading1--docs)   | A primary heading for main titles.             |
| [Heading2](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-typography-heading2--docs)   | A secondary heading for section titles.        |
| [Heading3](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-typography-heading3--docs)   | A tertiary heading for subsections.            |
| [Heading4](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-typography-heading4--docs)   | A smaller heading for additional sections.     |
| [Heading5](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-typography-heading5--docs)   | A minor heading for less prominent sections.   |
| [Heading6](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-typography-heading6--docs)   | The smallest heading for detailed subsections. |
| [Paragraph](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-typography-paragraph--docs) | A standard text container for body copy.       |
| [Blockquote](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-blockquote--docs)          | A styled container for quotes.                 |
| [HorizontalRule](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-horizontalrule--docs)  | A horizontal line for separating content.      |

#### Cards

Cards are available for separating content into well-defined sections. Easily use [swatches](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-themes--docs#swatch-colors) from the theme's color scheme to customize the look.

| Component                                                                                                                 | Description                         |
| ------------------------------------------------------------------------------------------------------------------------- | ----------------------------------- |
| [Card](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-cards-card--docs)             | A container for card-based content. |
| [CardHeader](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-cards-cardheader--docs) | A header component for card titles. |
| [CardBody](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-cards-cardbody--docs)     | A body component for card content.  |

#### Tables

| Component                                                                                                                | Description                                                        |
| ------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------ |
| [Table](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-tables-table--docs)         | A structured container for organizing data in rows and columns.    |
| [TableRow](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-tables-tablerow--docs)   | Represents a single row within a table.                            |
| [TableCell](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-tables-tablecell--docs) | A cell within a table row, used to display individual data points. |

| [CircleShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-circleshape--docs)       | A component for rendering SVG circles.    |
| [EllipseShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-ellipseshape--docs)     | A component for rendering SVG ellipses.   |
| [RectangleShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-rectangleshape--docs) | A component for rendering SVG rectangles. |
| [SquareShape](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-components-shapes-squareshape--docs)       | A component for rendering SVG squares.    |
