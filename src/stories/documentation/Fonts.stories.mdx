<h1>Fonts</h1>

## Overview

The underlying framework, [react-pdf](https://react-pdf.org/fonts), includes three basic fonts: `Courier`, `Helvetic`, and `Times-Roman`, with no **bold** or *italic* formatting enabled by default.

Most likely, you'll want to use fonts in your PDF that are beyond these basic choices, especially if you'd like to use formatting styles.

React PDF Builder includes support for 25 different fonts, with bold and italics, available via [jsdelivr](https://www.jsdelivr.com/) so you can get up and running quickly.

## Using built-in fonts

Fonts must first be registered with [react-pdf](https://react-pdf.org/fonts) to be usable in your PDF. 

Register any of the built-in fonts like so:

```ts
Font.register(Fonts.load(Fonts.sansSerif.roboto));
```

You can then style elements with the font family:

```tsx
<Div style={{ fontFamily: "Roboto" }}>This text has the Roboto font family.</Div>
```

## Supported fonts

React PDF Builder includes a selection of the most popular sans-serif, serif, and monospaced Google fonts, all curated from the [Google Webfonts Helper](https://gwfh.mranftl.com/fonts).

All built-in typefaces include the Latin character set, and include all supported font weights and styles, allowing you to use `fontWeight` and `fontStyle` styles, or the `fw-bold` and `fst-italic` [utility class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-utility-classes--docs#text), to format text.

## Using custom fonts

Looking to use a font that's not included with React PDF Builder?

To register a custom font, you'll use the [Font registration interface](https://react-pdf.org/fonts) via react-pdf.

At the moment, only TTF and WOFF font files are supported.

> *Side note: While building React PDF Builder, support for handwriting and display fonts was initially planned, but all of these fonts (20 total attempted) failed to load. In fact, 50 total fonts were planned, but half of them did not load. So your mileage may vary when loading custom fonts. Please [open a PR](https://github.com/justinmahar/react-pdf-builder/pulls) if you find a way to support handwriting and display fonts!*

### Step 1 - Upload TTF files to host

First, you will need to make sure all of your font files (TTF or WOFF) are at a publicly accessible URL. 

You can use the [Google Webfonts Helper](https://gwfh.mranftl.com/fonts) to download the appropriate font files:

1. Select your desired font
1. Select your desired charsets (choose Latin for English)
1. Select Legacy Support to download TTF files, or Historic Support to download WOFF files
1. Download, unzip, and upload the TTF or WOFF font files to your host

### Step 2 - Register custom font

Next, you'll [register the font](https://react-pdf.org/fonts) with react-pdf, being sure to specify the font weight and style for each font file:

```ts
Font.register({
  family: "Ubuntu",
  fonts: [
    { src: "https://example.com/fonts/ubuntu-v20-latin-300.ttf", fontWeight: 300 },
    { src: "https://example.com/fonts/ubuntu-v20-latin-300italic.ttf", fontWeight: 300, fontStyle: "italic" },
    { src: "https://example.com/fonts/ubuntu-v20-latin-regular.ttf", fontWeight: 400 },
    { src: "https://example.com/fonts/ubuntu-v20-latin-italic.ttf", fontWeight: 400, fontStyle: "italic" },
    { src: "https://example.com/fonts/ubuntu-v20-latin-500.ttf", fontWeight: 500 },
    { src: "https://example.com/fonts/ubuntu-v20-latin-500italic.ttf", fontWeight: 500, fontStyle: "italic" },
    { src: "https://example.com/fonts/ubuntu-v20-latin-700.ttf", fontWeight: 700 },
    { src: "https://example.com/fonts/ubuntu-v20-latin-700italic.ttf", fontWeight: 700, fontStyle: "italic" },
  ],
});
```

Generally speaking, "regular" means a font weight of `400`.

### Step 3 - Specify fontFamily style

Finally, specify the value you used for `family` as the `fontFamily` style for your PDF elements to use your custom font.

If you loaded font files supporting bold and italics, you can use the `fontWeight` and `fontStyle` styles, or the `fw-bold` and `fst-italic` [utility class names](https://justinmahar.github.io/react-pdf-builder/?path=/docs/documentation-utility-classes--docs#text), to format text.

```tsx
<Paragraph style={{ fontFamily: "Ubuntu" }}>
  ...
</Paragraph>
```
